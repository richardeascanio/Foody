# Foody

Modern Food Recipes App - Android Development with Clean Architecture

![alt text](https://i.postimg.cc/6pt0GT54/Thumbnail-1.png)

# Elements used
- Kotlin
- MVVM Clean Architecture
- Navigation Component
- Dagger - Hilt Dependecy Injection
- Room
- Glide
- Retrofit2
- Spoonacular API
- Data Binding
- Data Store
- View Binding
- Motion Layout
- Dark Theme
and others.
